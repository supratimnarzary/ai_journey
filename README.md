# 🤖 My AI Learning Journey

A project-based path to understanding AI deeply — built from easy to complex, where **every step is
documented and every concept explained in plain language**. Core ideas are deliberately repeated across
projects so they're drilled in for good.

## The roadmap

| # | Project | What it teaches | Builds toward |
|---|---------|-----------------|---------------|
| 1 | **House Price Prediction** | The full ML workflow on simple tabular data: framing, EDA, train/test split, leakage, training, evaluation, overfitting | Foundations |
| 2 | **Image Classification (CNNs)** | Neural networks, layers, activations, gradient descent, training loops | Computer vision |
| 3 | **OCR — Reading Text from Images** | Combining vision + sequence models to turn images into text | *Your OCR goal* |
| 4 | **Text Classification with Transformers** | Tokenisation, embeddings, attention, transfer learning / fine-tuning | Language models |
| 5 | **RAG Q&A System** | Embeddings, vector search, prompting, grounding an LLM in documents | Bridge to agents |
| 6 | **AI Agent with Tools** | LLMs that plan, use tools, and remember — the ReAct loop | *Your agents goal* |

> Every project reuses the same backbone — *frame → explore → split → preprocess → train → evaluate →
> improve* — so you see the same concepts again and again with new data and new models.
